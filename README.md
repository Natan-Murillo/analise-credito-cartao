# 📊 Análise de Inadimplência em Cartões de Crédito

Análise exploratória de um dataset com 30.000 titulares de contas de cartão de crédito.

## 📁 Arquivos
- `analise_credito_cartao.ipynb` — Notebook com toda a análise documentada
- `default_credit_card_clients.csv` — Dataset utilizado

## 🔍 O que foi feito
- Exploração inicial dos dados (colunas, linhas, tipos)
- Identificação de variáveis categóricas e numéricas
- Análise de distribuições e frequências
- Verificação de dados faltantes (explícitos e implícitos)

## 🛠️ Tecnologias
Python · Pandas · NumPy · Jupyter Notebook
